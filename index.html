SYS-01

HelioSentinel: Space-Weather Pipeline with an AI Narration Layer

Continuously ingests real solar-wind, proton-flux, event, and satellite data, flags dangerous conditions with deterministic rules, and uses a local language model to turn each anomaly into a plain-English incident report, graded against NASA's official event log.

NOAA · NASA · CelesTrak APIs → DuckDB → deterministic detector → local LLM (schema-validated) → eval harness / CI gate → FastAPI 3D console

DWG FL-001 · space-weather detect → narrate SCHEMATIC
On GitHub since 2026-07-12

Hardest decisionThe model never decides whether something happened; deterministic rules do. The LLM only describes and structures what happened. Why the model is kept out of the detection path →

+

Source connectors sit behind one plugin interface with strict rate-policy compliance; the deterministic detector is a set of reusable DuckDB window queries (same SQL for backtest and live). The local LLM's output is schema-validated with retry-on-error and failure quarantine, so malformed output never reaches the store. A hand-labeled golden set plus DONKI correlation gate every prompt change in CI; one documented iteration moved severity accuracy from 37% to 83%.

The operator console is a live 3D geospace situation display: Earth's magnetopause compresses from real solar-wind pressure and satellites are SGP4-propagated from real orbital elements, with every panel labeled LIVE (real physics / telemetry) or ILLUSTRATIVE (decorative).

SYS-02

MLB Analytics Pipeline

Takes 2.18M raw baseball pitch records and turns them into 13 statistical models and a live dashboard, rebuilt from scratch every night, automatically, with the build refusing to ship if any check fails.

Statcast → DuckDB → dbt (Kimball, SCD2) → R + Python models → static dashboard

DWG FL-002 · Statcast → parity-gated models SCHEMATIC
On GitHub since 2026-07-04

Hardest decisionEvery model is implemented twice (once in R, once in Python) and deployment is blocked unless both agree. Here's why the duplication is the feature →

+

A tested bronze / silver / gold medallion with a Kimball star schema and a daily-grain SCD2 dimension. The run-expectancy matrix (RE24) is rebuilt from scratch; the win-probability model is evaluated once on a sealed 2025 holdout and beats both baselines.

A separate live 2026 layer refreshes playoff odds nightly while the validated analysis stays frozen, surfaced in an audience-oriented dashboard with an honest, uncertainty-forward voice.
